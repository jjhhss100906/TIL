# css반응형 디자인이란?

 반응형 디자인은 다양한 기기들이 나오고,다양한 화면크기가 나오면서,**각각다른 화면에따른 설계방식**이 필요해져서 나온 개념이다.

여기서 화면크기는 브라우저의 크기,즉 **viewport**를 의미한다.
만약 pc에서 보는 가로페이지를 스마트폰에 띄우면 맞지 않을것이고,
모바일 환경은 해상도도 pc보다 떨어지기 때문에 웹환경을 변경하지 않고 그대로 이식하는 것은 무리이다.   

이것을 해결하기 위해 나온것이 **반응형 디자인**이라는 것이다.

## 반응형과 적응형 차이

반응형은 하나의 웹페이지가 여러 디바이스의 **크기에 맞게 조절**되는 반면에,적응형은 디바이스마다 적절한 웹페이지를 **따로따로 만드는것**을 뜻한다.   
즉,반응형과 적응형은 **웹페이지를 디자인하고 렌더링하는 방식의 차이**인 것이다.

## 미디어 쿼리(Media Query)

사이트에 접속하는 장치에 따라 특정한 css스타일을 사용하도록 도와주는 소프트웨어 모듈   

1. @media 규칙을 통해 서로 다른 매체 유형(media type)을 위한 맞춤식 스타일 시트를 지원한다. 여기서@media는 **조건문 시작 키워드**라고 보면 된다.        
2. 미디어 쿼리는 width,height,color 속성과 같은 미디어 관련 속성을 이용한 **표현식**을 통해 **스타일이 적용되는 범위를 조절한다.    
3. 콘텐츠를 별도로 변경하지 않아도 **웹 페이지에 접속하고 있는 기기에 알맞은 형태로 스타일이 조정**된다.여기서 콘텐츠는 웹페이지 안에 들어있는 내용물 자체를 말한다.   

## 미디어 쿼리 구문

```
@media media-type and|only|not (media-feature-rule) {
  /* 여기에 css 스타일 코드 입력 */
}
```
* 웹페이지에 접속하고 있는 기기의 **미디어 유형**과 구문에 명시된 ```media-type``` 이 일치하고, 모든 ```media-feature-rule``` 이 참이면 미디어 쿼리는 참을 반환하게 되어 해당 블록 안에 명시된 CSS 스타일 코드가 실행된다.   

## media type: 미디어 유형

* all: 모든 미디어 유형
* print: 컴퓨터 스크린
* screen: 컴퓨터나 테블릿,스마트폰 등 스크린이 있는 매체에 사용한다.   
* speech: 웹 페이지를 읽어주는 스크린 리더에 사용한다.스크린 리더는 화면에 있는 내용을 읽어서 들려주는 프로그램이다.

## and|only|not

더욱 복잡한 조건을 명시하기 위한 연산자 키워드   

* and:미디어 쿼리 조건을 추가한다.
* only:미디어 쿼리가 호환되는 브라우저에만 인식 가능하게 한다.
* not:not 뒤에 지정하는 미디어 유형은 제외한다.

## 미디어 쿼리 속성

* width:화면의 너비
* height:화면의 높이
* device-width:매체 화면의 너비
* device-height:매체 화면의 높이
* devie-aspect-ratio:매체 화면의 비율
* orientation:매체 화면의 방향(min,max 사용불가)
* color:매체의 색상 비트 수
* color-index:매체에서 표현가능한 색상의 개수
* monochrome:흑백 매체에서의  픽셀당 비트 수
* resolution:매체의 해상도

## 너비와 높이

* width 및 height미디어 기능은 **범위 지정**에 사용될 수 없다.
* min- 또는 max- 접두사를 붙이게 되면 최소값과 최대값을 표시할수 있다.
```
@media (orientation: landscape) {
    body {
        color: blue;
    }
}
```
이렇게 하면 화면 영역의 크기가 400픽셀보다 좁은 경우에는 색깔을 파란색으로 하라는 말이 된다.

## 방향성

* orientaion로 세로 모드인지 가로 모드인지 검사가 가능하다.
```
@media (orientation: landscape) {
    body {
        color: blue;
    }
}
```
이렇게 하면 가로 모드일때 색깔을 파란색으로 하라는 말이 된다.
* landscape: 가로 화면
* portrait:세로 화면
