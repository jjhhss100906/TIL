# css grid layout

아까 정리한 flex박스는 가로또는 세로중 하나만 다루는 방식인 1차원인 반면에 grid박스는 가로와 세로를 둘다 다루는 2차원 방식이다.

## 그리드 display 속성

* ``` grid-template-columns```:컬럼지정
* ``` grid-template-row```:줄지정

## 상대적인 크기를 지정하는 fr단위

* grid레이아웃에서 픽셀단위를 사용하면 항상 고덩된 크기를 가지므로 반응형 웹 디자인에는 맞지 않다 그래서 상대적인 크기를 지정하는 fr단위를 사용한다.   
* 같은 단위의 반복일때는 repeat를 사용해도 된다.

## minmax()함수

* minmax()함수를 사용하면 유연하게 줄 높이의 최소값과 최대값을 지정할수있다

## 자동으로 column의 개수를 지정하는 auto-fill,auto-fit

* repeat()함수의 ()안에 들어가는 열과 행을 지정하지 않고, 공간이 될때까지 채운다.
* auto-fill은 빈공간이 있으면 남기고,auto-fit은 빈공간을 없애고 늘려버린다.

## 그리드 항목의 간격 지정

* grid-column-gap:열 사이의 간격을 지정한다.
* grid-row-gap:행 사이의 간격을 지정한다.
* grid-gap: 한번에 둘다 설정한다.
참고로 요즘은 gap하나로 통일되었다.

## 그리드 라인을 이용해 배치하기


  그리드 레이아웃은 눈에 보이지 않는 그리드 라인이 포함되어있는데, 종류는 이와 같다

  * grid-column-start:열 시작 라인의 번호를 지정한다.
  사용할때는 start뒤에 :숫자 를 쓰면된다.
  * grid-column-end:열 마지막 라인의 번호를 지정한다.
  사용할때는 end뒤에 똑같이 :숫자 를 쓰면된다.
  * grid-column:열 시작번호와 끝번호 사이에 /를 넣엇서 사용함.
  사용할때는  column뒤에 :(시작할 라인의 번호)/(마지막 라인의 번호)를 넣어서 사용한다.
  * grid-row-start:줄 시작 라인의 번호를 지정한다.
  사용할때는 위와 같이 :숫자 를 사용한다.
  * grid-row-end:줄 마지막 라인의 번호를 지정한다.
  사용할때는 :숫자 를 사용한다.
  * grid-row:줄 시작 번호와 줄 끝 번호 사이에 /를 넣어 사용함.
  사용할때는 grid-column처럼 (시작 러인 번호)/(끝 라인 번호)를 넣어서 사용한다.

  ## 템플릿 영역을 만들어 배치

  * grid-area:각 영역에 이름을 붙히고 만약 부모영역을 #wrapper로 해놓았다고 치면 그 안에 grid-template-areas 속성을 사용해서 영역 배치를 지정한다.   
  * 만약 템플릿 영역을 비우고 싶다면 그자리에 (.)을 넣으면 된다.
  * 한줄에 들어갈 템플릿 영역은 ("템플릿")처럼 큰 따옴표로 묶어준다.
